# ExamenEntornos

README
Gestión de Garaje
Este programa permite gestionar vehículos en un garaje con las opciones:

Agregar vehículos.
Mostrar vehículos.
Eliminar vehículos por matrícula.


Cómo ejecutar:

Compila el programa.
Ejecuta y sigue el menú interactivo.
Propósito:
Practicar optimización de código y análisis de tiempos.
